# NorthStar Advisory

A modern, premium **consulting firm landing page** built using **HTML, CSS, and vanilla JavaScript**, designed to showcase professional UI design, smooth animations, and interactive user experiences.

---

## Project Overview


**NORTHSTAR ADVISORY** is a fully responsive, high-end corporate landing page built with HTML, CSS, and vanilla JavaScript. It features a clean enterprise design with animated hero statistics, smooth scrolling navigation, interactive service cards, testimonial video sections, case study showcases, downloadable resources, blog previews, and a functional consultation booking form.

The project focuses on **professional front-end architecture**, **modern design systems** using CSS variables, **smooth micro-interactions**, and structured DOM manipulation — all **without using external JavaScript frameworks**.

This project demonstrates how to build a scalable, production-style corporate website using pure front-end technologies.

> **Note:**  
> This is a personal portfolio project created for learning and demonstration purposes.

---

## Features
> **Note:**  
> Some features are currently being made.


- Fully responsive design (desktop, tablet, mobile)
- Fixed navigation with scroll-based shadow effect
- Smooth scrolling with dynamic offset handling
- Animated hero statistics counter (Intersection Observer)
- Services grid with hover animations
- Appointment booking form with validation
- Future-date enforcement for consultations
- LocalStorage simulation for appointment saving
- Case study cards with animated hover effects
- Case study download simulation with tracking
- Embedded testimonial video section
- Blog preview cards
- Scroll-to-top button with visibility logic
- Intersection Observer-based scroll animations
- Modular JavaScript architecture
- Utility functions (debounce, throttle, viewport detection)

Structured CSS design system using variables
---

## Tech Stack

- **HTML5** — Semantic layout and structured sections  
- **CSS3** — Design system with variables, modern layouts, animations, responsive grid
- **JavaScript (Vanilla)** — DOM manipulation, form validation, localStorage handling, scroll logic, animations

No external frameworks or libraries are used (except icon fonts and web fonts).
